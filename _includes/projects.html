Projects

A-Paper-A-Week - Committed myself to reading and summarising a research paper every week.
CNN-Sentence-Classifier - Keras based implementation of Convolutional Neural Networks for Sentence Classification by Yoon Kim (2014).
Character-Level Language Model - TensorFlow based implementation of Char-RNN using Multi-layer Recurrent Neural Networks.
Spatial Co-location Pattern Mining using Graph Database - Developed and implemented scalable algorithms for mining spatial co-location patterns using distributed graph database as part of the final year thesis.
MapReduce System to mine Spatial Co-location Patterns - Efficient implementation of the system described in the paper A Parallel Spatial Co-location Mining Algorithm Based on MapReduce by Yoo et al (2014)
Stack Exchange Data as Spark Dataframes - Implemented a system that loads Stack Exchange Data as Spark Dataframes and performs analysis over the data.
Tagging StackExchange Questions using SVC - Implemented a tagger for questions posted on Stack Exchange using Support Vector Classification as course project for Artificial Neural Networks, IIT Roorkee.
Analysing Power-law Distributions - Python implementation of the system described in the paper Power-law distributions in Empirical data by Clauset et al (2009).
